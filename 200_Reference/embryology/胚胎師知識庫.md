## 備盤

### 概念說明

| 名稱 | 時間點 | 執行者 | 目的 |
|---|---|---|---|
| 估盤 | 前一天下班前 / 當天早上 | 抓盤人（2 位）| 根據隔天手術台數估算盤子數量與培養液用量 |
| 備盤 / 點盤 | 當天下午 1–2 點 | 備盤人（2 位）| 實際配置培養液（Medium），記錄批號與開封日期 |

雙人核對機制：
- 估盤完成後由第二人複核，有異動需兩人共同確認
- 點盤時兩人同步操作，再次確認盤數與估盤一致

### 一、GIVF（G-IVF）卵子培養液

GIVF 適用盤子：所有卵子類相關盤（取卵、解卵、AOA、精蟲操作）

**GIVF 總量公式（Excel/Sheets）：**
```
GIVF 總量 = OPU盤 × 3 + (De + TE + AOA)盤 × 0.5 + 精蟲(mL) + 多配(mL)
```

#### 各盤說明

**1. OPU 盤（取卵盤）**
- 數量 = 取卵台數
- 多備條件（Organ 標準，由抓盤胚胎師評估）：
  - AMH ≧ 2
  - NS（E2 > 2000）
  - 且：凍卵 T2O ≧ 36.5℃ 以上 / 受精 T2O ≧ 37℃ 以上（或卵非常多）
  - 滿足以上條件時，根據明天人力安排及卵數分配，決定是否增備
- GIVF 用量：**3 mL / 盤**

**2. De 盤（Denuding 盤，去除卵丘細胞）**
- 數量 = 取卵台數（固定，不額外增加）
- GIVF 用量：**0.5 mL / 盤**

**3. TE 盤（解卵盤）**
- TE = 解凍卵子（解卵），兩者意思相同
- 數量 = 解凍卵子台數
- 增加 1 盤的情況：
  - (a) 解凍顆數或支數較多時
  - (b) 102（解凍試劑）到貨，當天要準備 QC 試驗組時
- GIVF 用量：**0.5 mL / 盤**

**4. AOA 盤（Assisted Oocyte Activation，輔助卵子活化）**
- 數量 = 有受精客戶數 + 1 盤備用
  - 備用盤是為了因應臨時追加 AOA 使用
  - 若所有受精客戶本來就都需要 AOA，則不再額外加備用盤
- 兩種 AOA 試劑（可互換）：
  - 弘優 CI（Calcium Ionophore）：常規使用
  - 明美 GM508：特殊標記客戶使用
- GIVF 用量：**0.5 mL / 盤**

**5. 精蟲(mL)**
- 用量 = 受精台數 × **0.7 mL**
- 注意：這是直接填 mL 數，不是盤數，直接累加進 GIVF 總量

**6. 多配(mL)（GIVF 區）**
- 最低 **2 mL**（放入培養箱平衡，防蒸發而設保底）
- 實際通常估 3–4 mL，根據受精台數加總：
  - ICSI 盤（顯微受精）：≈ **50 μl / 台**
  - Standard 盤（自然受精）：≈ **500 μl / 台**（約 8 顆卵子 / 盤）
  - Spindle 盤：≈ **50 μl / 台**
- 注意：這是直接填 mL 數，直接累加進 GIVF 總量

### 二、胚胎培養液（H5GT + GxTL）

**現階段主力**：H5GT
**輔助**：GxTL（主要用於雙線培養，詳見 Geri-受精說明）

**基礎用量（大多數盤子）：**
- H5GT：**0.4 mL / 盤**
- GxTL：**0.4 mL / 盤**

**GxTL 合計公式（Excel/Sheets）：**
```
GxTL 合計 = (D1 STD + D3 + D3+Bx + Bx + TB + Glue + Geri-FBT + 多配盤數 + D0) × 0.4
           + Geri-受精 GxTL 盤數 × 0.4
```

**H5GT 合計公式（Excel/Sheets）：**
```
H5GT 合計 = (D0 + D3 + D3+Bx + Bx + TB + Glue + Geri-受精 H5GT + Geri-FBT + 多配盤數) × 0.4
```

> 雙線培養時，同一客戶有一盤 H5GT + 一盤 GxTL，因此表格採欄位矩陣（B 欄 = GxTL 盤數，C 欄 = H5GT 盤數）。

#### 各盤說明

**1. Geri-受精盤（縮時攝影培養箱受精盤）**
- 數量 = 受精客戶數
- 雙線培養：同一客戶若成熟卵 > 8 顆，有機會安排雙線培養（一盤 H5GT + 一盤 GxTL）
  - 是否雙線培養：需查閱隔天客戶資料（客戶 review），依個案評估
- 培養液：**H5GT 0.4 mL / 盤；GxTL 0.4 mL / 盤**（雙線時各備一盤）

**2. D0 盤（Day 0 乾式培養箱備案盤）**
- 使用時機：Geri 縮時攝影培養箱機台已滿，部分客戶改用 D0 盤 + 乾式培養箱培養
- 培養規則與 Geri-受精盤相同
- 培養液：**0.4 mL / 盤**
- 彈性備用邏輯：備 D0 盤時，多配(mL)中額外預留 0.4 mL，供隔天若培養箱有空位時緊急新配一盤 Geri 盤

**3. Bx 盤（Biopsy 採樣盤）**
- 目前主要使用 **H5GT**
- 增加 1 盤的情況：
  - (a) 受精後第 5 天的客戶，需要進行 PGS（胚胎基因採樣）
  - (b) 客戶不進行採樣，但卵子顆數 > 5 顆時，多備 1 盤
- 培養液：**0.4 mL / 盤**

**4. TB 盤（解配盤 / 解凍植入盤）**
- 數量 = FBT（Frozen-thawed Blastocyst Transfer，解凍植入）台數
- 培養液：**0.4 mL / 盤**

**5. Geri-FBT 盤（解凍胚胎恢復觀察盤）**
- 解完胚後，胚胎移入 Geri-FBT 盤做恢復觀察
- 數量 = FBT 台數（若培養箱位置有限，可視情況取捨）
- 培養液：**0.4 mL / 盤**

**6. Glue 盤（EmbryoGlue 轉盤，植入盤）**
- 恢復完成後，胚胎移入 Glue 盤（又稱轉盤）
- 數量 = FBT 台數；若有新鮮植入（Fresh ET），額外 +1 盤
- 培養液：**0.4 mL / 盤**

**7. 多配(mL)（胚胎培養液區）**
- 最低 **2 mL**（防蒸發保底）
- 主要用途：
  - Biopsy 操作盤
  - 臨時增配 Geri 盤（當 D0 盤客戶隔天有機會住進 Geri 時）
  - 受精盤數量不足時的臨時備盤
- 預留邏輯：若當天有備 D0 盤，額外預留 **0.4 mL** 作為緊急新配 Geri 盤使用

**8. D1 STD / D3 / D3+Bx（目前暫停使用）**
- D0 天 = 受精當天；D1 = 隔天；D3 = 第 3 天，以此類推，最多到 D7
- 這些盤是胚胎發育到對應天數時使用的培養盤，目前無需備盤
- 系統設計原則：**目前不顯示**；未來若需使用，可手動新增盤子種類（含名稱、規則、用途）

### 三、HE（Heat Equilibration，培養液預熱瓶）

**HE 估算公式（Excel/Sheets）：**
```
HE 瓶數 = MAX(8, ROUNDUP(≦5台數 × 0.5 + 6-9台數 × 1 + ≧10台數 × 2 + 梁+金台數 × 1 + 卵多的賴台數 × 1 + 1, 0))
最低保底：8 瓶（2 間培養箱，每間至少 4 瓶）
```

輸入欄位（估算時需填入）：
- ≦5 台的手術幾台
- 6–9 台的手術幾台
- ≧10 台的手術幾台
- 梁+金 幾台
- 卵多的賴 幾台

### 四、Oil（覆蓋油）

**Oil 估算公式（Excel/Sheets）：**
```
Oil 份數 = ROUNDUP(((總受精數 ÷ 3) × 7 + (spindle盤 + Bx盤) × 5 + STD台數 × 10 + 50) ÷ 50, 0)
```

輸入欄位（估算時需填入）：
- 總受精數（sona + 解卵數）
- STD 台數（Standard 受精，若顆數很多要多估）
- spindle 幾盤
- Bx 幾盤

### 五、培養液開封後效期規則

| 培養液 | 開封後效期 | 計算方式 |
|---|---|---|
| 大多數培養液（GIVF、GxTL、H5GT 等）| **7 天** | 開封日算第 1 天，第 7 天為最後可用日 |
| CIR / AOA（宏優 Calcium Ionophore）| **14 天** | 同上 |
| GLUE（EmbryoGlue）| **14 天** | 同上 |

**計算範例：**
- 開封日 3/24（第 1 天）
- 效期 14 天 → 第 14 天 = 4/6（最後可用日）
- 4/7 起不可使用

### 六、備盤培養液批號規則

- **同批號可混用**，不同批號的培養液不可混入同一個盤子
- 多批號同天情境：舊批用完後開新批補充，是正常的批次切換，不是兩批同時倒入同一盤
- FIFO 原則：依開封日最早的批次優先使用
