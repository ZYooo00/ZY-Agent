## /ultrareview 是什麼？

這是 Claude Code 的**雲端深度代碼審查**功能。你在 CLI 輸入 `/ultrareview`，它會：

- 把你目前 branch 的修改**上傳到雲端沙箱**
- 同時派出**一群 AI reviewer 平行審查**
- 每個發現的 bug 都會**獨立驗證**，才回報給你（減少誤報）
- 全程在遠端跑，你的終端機可以繼續做其他事

## 跟 /review 有什麼差別？

| | `/review` | `/ultrareview` |
|--|--|--|
| 執行位置 | 本機 | 雲端沙箱 |
| 深度 | 單輪審查 | 多代理人 + 交叉驗證 |
| 時間 | 幾秒到幾分鐘 | 約 5–10 分鐘 |
| 費用 | 算在正常用量 | 免費次數用完後 $5–$20 |
| 適合時機 | 邊開發邊快速確認 | **要 merge 前的最終把關** |

## 你需要用這個嗎？

就你目前的工作（培養液管理系統的 HTML 頁面）來說：
- 這個功能更適合**多人協作的 Production 代碼**或**大型 PR 合併前的審查**
- 你目前的 HTML/JS 專案規模，`/review`（本地）已經夠用，不需要動用 ultrareview

**但如果你今天有用 Pro/Max 訂閱，可以趁免費次數到期前試用看看**，感受一下差異，純粹體驗性質。
